# Medical Categories Component

A responsive "Categories" UI component for a medical home page built with React and Tailwind CSS.

## Features

- Responsive grid layout with 4 columns on desktop, adapting to smaller screens
- Clean, professional medical icons for each category
- Squircle-shaped buttons with rounded corners
- Truncated text with ellipsis for long category names
- Dark blue header text with semi-bold font
- Muted gray category labels

## Categories Included

- Psychiatrist (Brain icon)
- Hepatologist (Liver icon)
- Cardiologist (Heart icon with heartbeat line)
- Dental (Tooth icon)
- Nephrologist (Kidneys icon)
- Pulmonologist (Lungs icon)
- Dermatologist (Skin layers/hair follicle icon)
- Gastroenterologist (Stomach icon)

## Usage

To run the component:

```bash
cd web_components/categories
npm install
npm start
```

Then open your browser to the provided URL to see the component in action.

## Component Structure

The component is built as a React functional component with the following features:

- Header with "Categories" title (dark blue, 24px semi-bold)
- Grid layout with 4 columns (responsive)
- Category cards with:
  - White background
  - Subtle gray border
  - Rounded corners (squircle shape)
  - Professional medical SVG icons
  - Centered, truncated category names in muted gray
